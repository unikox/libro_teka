# libro_teka
Symfony 5.x API Library (testing)

Для установки и запуста данного проекта вам потребуется Docker
-------------------
```
Установка
    Скачиваем проект:
    git clone https://github.com/unikox/libro_teka.git  libro_teka 

Запуск
    для запуска нужно выполнить комманду:
    cd ./libro_teka/local
    sh start.sh

Использование
    Спустя некоторое время, в зависимости от мощности вашего пк, запустятся 2 контейнера. 
    Внешний https порт 4499 контейнера, используется nginx с самоподписными сетификатами так что предупреждение можно смело пропускать
        swager доступен по ссылке: https://127.0.0.1:4499/api/doc
    Внешний mysql порт 5306 контейнера, используется mysql:8.0
        для подключения к бд 
                user:root 
                password:secret

Отладка
    Данный проект сконфигурирован на использование Xdebug и VSCode
        в VScode используется расширение PHP Debug v1.26.0
        сам Xdebug 3.x

Остановка
    Для остановки проекта выполните: 
        sh start.sh 
        в дириктории local текущего проекта
```
